# 恢复 Windows 11 传统完整右键菜单

Windows 11 默认右键菜单被简化，若要恢复类似 Windows 10 的完整菜单，可通过注册表修改实现。

## 方法一：使用 CMD 命令永久恢复

1. **以管理员身份运行命令提示符**  
   - 右键点击屏幕底部的 **“开始”** 图标，选择 **“终端（管理员）”** 或 **“命令提示符（管理员）”**。

2. **执行注册表命令**  
   输入以下命令并回车（可直接复制）：
   ```cmd
   reg.exe add "HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}\InprocServer32" /f /ve
   ```

3. **重启资源管理器**  
   - 方法一：在任务管理器中找到 **“Windows 资源管理器”**，右键选择 **“重新启动”**。  
   - 方法二：直接重启电脑。

## 方法二：快捷键临时显示完整菜单

若仅需临时查看完整右键菜单，可：
- 点击右键菜单底部的 **“显示更多选项”**。
- 或直接使用快捷键 **`Shift + 右键`**。

## 恢复 Windows 11 默认简化菜单

如需改回默认样式，以管理员身份运行命令提示符，执行：
```cmd
reg.exe delete "HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}" /f
```
然后重启资源管理器。

## 注意事项

- 通过工具或修改注册表均可自定义右键菜单。
- 如果右键功能出现混乱，可尝试使用系统自带的 **“安全工具”** 检查。
- 若不习惯鼠标左右键操作，可在 **“设置 -> 设备 -> 鼠标”** 中进行调整。
